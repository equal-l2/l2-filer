# l2-filer
A primitive filer written in Rust.

## How to use
- Choose item with Up/Down key.
- Change page with Left/Right key.
- Edit file or change directory with Return key.
  (You can be set the editor with `EDITOR` environmental variable)
- Quit with q key.
